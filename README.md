# Zoma2 
Working Repository of Zoma2 for a Software Engineering Project

# Usage
## Server
 - Install NodeJS & MongoDB
 - Start the mongodb service
 - clone this repository `git clone https://github.com/rohinrohin/Zoma2`
 - `cd zoma2` 
 - `npm install`
 - `npm run debug`
